# Espaco
<--!Partindo de como fazer a inicialização do projeto: 1- Precisa ter instalado o Node.js e o Git para clonar. 2- No terminal você irá executar esses comandos: cd backend
npm install. 3- Espaços padrões no firebase: cd backend
node seed.js, 4- Para iniciar o backend vá para o terminal e coloque os comandos: cd backend
npm start ou npm run dev, dai você irá receber a seguinte mensagem "Servidor rodando na porta 5000", essa foi a parte do backend, já a parte do front-end vá até a pasta frontend/public, abra o arquivo index.html na extensão baixada no VS Code chamada de Live Server (Forma mais fácil)! Mas também tem a opção de entrar pelo Python: cd frontend/public
python -m http.server 8000, e a opção do Node.js (http-server): npm install -g http-server
cd frontend/public
http-server.
